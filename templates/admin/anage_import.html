AnAge lifespan data import


This page allows you to import AnAge data (CSV format) into MitoAge.

Upload a comma seperated text file (CSV) below and submit.
Following the upload, you will be able to review the data.


Data format


Data must be in a CSV format with 3 columns specifying the name of the species, its lifespan (in years) and the HAGR ID. The header of the columns should specifically be: "Species", "Lifespan" and "HAGRID". Lifespan will be specified in years.
