Taxonomy data import


This page allows you to import taxonomy data (CSV format) into MitoAge.

Upload a comma seperated text file (CSV) below and submit.
Following the upload, you will be able to review the data (errors, duplicate or incomplete lines, entries already in the DB, new entries to be added).


Data format


Data must be in a CSV format. Name of species is mandatory; families, orders and class are all optional. Common name is optional.
Although some fields might be optional, the first line of the file is mandatory and should contain the header with all 5 columns (please see example below).
